class DoublyLinkedList.DoublyLinkedList¶

Bases: object

A collection of nodes which is bi-directionally connected.
The class comes equipped with a Constructor and methods such as printer, reverse, insert
Parameters:

  • tail – Leading Node of the Doubly Linked List

printer(sep=', ')¶
Prints the Doubly Linked List in a visualizable format
Parameters:

sep (string, optional) – Delimits adjacent element terms, default value = ‘, ‘
Example:
>>> S = DoublyLinkedList()

>>> S.insert(50)

reverse()¶
Reverses the order of the elements of the Linked List
Swaps head and tail
Example:
>>> S = DoublyLinkedList()
>>> S.insert(60)
>>> S.insert(50)
>>> S.insert(70)
>>> S.reverse()
