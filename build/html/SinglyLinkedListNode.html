SinglyLinkedListNode module¶

class SinglyLinkedListNode.SinglyLinkedListNode(data)¶

Bases: object

This implements a node of a Singly Linked List.
The node consists of data and a next node element.
This implementation allows for the uni-directional nature of a singly linked list.
It comes with a constructor and a converter.
Parameters:
  • data (int) – The data or contents carried by the node.
  • next (SinglyLinkedListNode) – Leads to the next element in the Linked List.
__init__(data)¶
Constructs a Node object with the given content/data
Parameters:

data – Initializing content for node.
Example:
>>> A = SinglyLinkedListNode(10)
>>> print(A.data)
10
__str__()¶
Convert Node to string format, ready to print
Returns:

string with data in self
Return type:

string
Example:

>>> print(str(S))
