BinarySearchTree module¶

class BinarySearchTree.BinarySearchTree¶

Bases: object

A Tree in which each node in the tree has two children.
The left child of the the node has a value less than the value of its parent
The right child of the node has a value greater than or equal to its parent
Has a Constructor, and methods for insert, traverse and height.
Parameters:root (BSTNode) – The only node with no parent. The ancestor of all nodes.
__init__()¶
Constructs a BInary Search Tree with empty root
Example:
>>> BST = BinarySearchTree()
>>> BST.root = BSTNode(56)
>>> print(BST.root)
56
height(root)¶
Gives the height off the tree
This is the distance between the root and the farthes descendant
The childless node has 0 height
Parameters:

root (BSTNode) – The root of the tree/sub-tree whose height is to be found
Returns:

height of tree, defined 0 for childless node
Return type:

int
Example:
>>> Y = BinarySearchTree()
>>> Y.insert(40)
>>> Y.insert(30)
>>> Y.insert(50)
>>> Y.insert(35)
>>> Y.insert(45)
>>> Y.insert(25)
>>> Y.insert(55)
>>> Y.height(Y.root)
2
insert(val)¶
Inserts a new node in the Binary Search Tree
Parameters:

val (any comparable object) – Value to be contained in the node
Example:
>>> BST = BinarySearchTree()
>>> BST.insert(28)
>>> BST.insert(73)
>>> print(BST.root)
28
>>> print(BST.root.right)
73
>>> print(BST.root.left)
None
traverse(order)¶
Traverses the tree in a fashion chosen by the member function
Offers option of PRE-ORDER, IN-ORDER and POST-ORDER traversal
Parameters:

order (string) – specifies the traversing fashion
Example:
>>> Y = BinarySearchTree()
>>> Y.insert(40)
>>> Y.insert(30)
>>> Y.insert(50)
>>> Y.insert(35)
>>> Y.insert(45)
>>> Y.insert(25)
>>> Y.insert(55)
>>> Y.traverse('PRE')
40 30 25 35 50 45 55
